# 中药材识别系统

> **基于卷积神经网络(CNN)的中药材图像识别 + Coze智能体信息查询**
>
> 算法设计与案例分析课程设计作品

## 项目简介

本项目是一个**中药材智能识别系统**，采用深度学习技术实现中药材图像的自动分类识别，并结合大语言模型智能体提供中药材相关的知识问答服务。

系统核心分为两部分：
1. **图像识别模块**：基于卷积神经网络(CNN)训练中药材分类模型，实现对上传图片的中药材种类识别
2. **智能查询模块**：调用Coze平台搭建的智能体API，根据识别结果自动查询中药材的性味、功效、用法等详细信息

## 核心功能

- **图像分类识别**：上传中药材图片，系统自动识别药材种类
- **多模型对比**：对比不同CNN架构（ResNet、VGG、MobileNet等）的识别效果
- **智能信息查询**：识别完成后自动调用Coze智能体API查询药材详细信息
- **交互式Notebook**：完整的Jupyter Notebook实验代码，包含数据预处理、模型训练、评估测试全流程

## 模块说明

### 1. 单模型训练 (`单模型训练.ipynb`)

- 数据预处理：图像增强、归一化、数据划分
- 模型构建：自定义CNN架构（卷积层+池化层+全连接层）
- 模型训练：训练过程可视化（损失曲线、准确率曲线）
- 模型评估：测试集准确率、混淆矩阵、分类报告

### 2. 多模型对比与智能体调用 (`多模型对比和智能体调用.ipynb`)

- **多模型对比**：
  - ResNet（残差网络）
  - VGG（深度卷积网络）
  - MobileNet（轻量级网络）
  - 自定义CNN
  - 对比指标：准确率、参数量、推理速度

- **Coze智能体API调用**：
  - 图像识别结果作为输入
  - 自动构建查询Prompt
  - 调用Coze API获取中药材详细信息
  - 返回药材名称、性味、功效、主治、用法用量等

### 3. 课程设计论文

完整的中药材识别系统设计与实现论文，包含：
- 选题背景与意义
- 相关技术综述（CNN、深度学习、图像识别）
- 系统需求分析与设计
- 模型构建与训练过程
- 实验结果与分析
- 总结与展望

## 模型性能

| 模型 | 准确率 | 参数量 | 特点 |
|------|--------|--------|------|
| 自定义CNN | ~85% | 较少 | 轻量级，适合嵌入式 |
| ResNet | ~92% | 较多 | 残差连接，训练稳定 |
| VGG | ~90% | 多 | 结构简单，层数深 |
| MobileNet | ~88% | 少 | 移动端友好，轻量化 |

> 注：具体性能指标以论文中实验数据为准
